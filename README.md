# 初中语文基础知识学习平台

基于《初中语文基础知识手册》开发的系统性语文基础知识学习网站。

## 功能模块

### 📚 知识学习
- 三级目录导航（编 → 部 → 节）
- 全文搜索（高亮显示）
- 拼音标注、TTS朗读
- 表格分页展示
- 收藏与笔记功能
- 学习进度追踪

### 🎯 刷题练习
- **12种题型**：字音、字形、词语运用、病句辨析、修辞手法、文学常识、文化常识、古诗文默写、名著阅读等
- **5种模式**：按知识点练习、随机抽题、模拟测试、错题重练、真题套卷
- **3个难度**：简单、中等、困难
- **2种来源**：预制题（AI 逐题编写）+ 中考真题（人工录入）

### ❌ 错题本
- 自动收集做错的题目
- 查看正确答案与解析
- 错题重练功能

### 🔍 全文搜索
- 搜索全部手册内容
- 关键词高亮显示
- 热门搜索推荐
