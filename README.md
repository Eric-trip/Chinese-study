# 初中语文基础知识学习平台

基于《初中语文基础知识手册》开发的系统性语文基础知识学习网站。

## 功能模块

### 📚 知识学习
- 三级目录导航（编 → 部 → 节）
- 全文搜索（高亮显示）
- 拼音标注、TTS朗读
- 表格分页展示
- 收藏与笔记功能
- 学习进度追踪

### 🎯 刷题练习
- **7种题型**：字音、字形、成语、修辞、文学常识、名句默写、病句
- **4种模式**：按知识点练习、随机抽题、模拟测试、错题重练
- **3个难度**：简单、中等、困难
- 题目基于手册数据自动生成

### ❌ 错题本
- 自动收集做错的题目
- 查看正确答案与解析
- 错题重练功能

### 🔍 全文搜索
- 搜索全部手册内容
- 关键词高亮显示
- 热门搜索推荐

## 技术栈

- 纯前端：HTML + CSS + JavaScript（无框架依赖）
- 数据：JSON文件加载
- 存储：localStorage（进度、错题、笔记、收藏）
- 部署：静态文件，可部署到任意Web服务器

## 项目结构

```
chinese-learning/
├── index.html          # 首页
├── knowledge.html      # 知识学习
├── practice.html       # 刷题练习
├── search.html         # 全文搜索
├── errorbook.html      # 错题本
├── css/
│   └── style.css       # 全局样式
├── js/
│   ├── data.js         # 数据加载与工具
│   ├── question-bank.js # 题库生成
│   ├── app.js          # 首页逻辑
│   ├── knowledge.js    # 知识页逻辑
│   ├── practice.js     # 练习逻辑
│   ├── search.js       # 搜索逻辑
│   └── errorbook.js    # 错题本逻辑
└── data/
    └── handbook.json   # 手册数据
```

## 扩展性设计

- **高中扩展**：数据结构和界面均预留高中学段扩展接口
- **用户系统**：当前使用localStorage，架构可升级为前后端分离
- **题库扩展**：question-bank.js 可独立扩展新题型

## 部署方式

将整个目录上传到任意Web服务器即可：

```bash
# 例如使用 Python 内置服务器
python3 -m http.server 8000

# 或使用 Nginx / Apache
# 将目录配置为静态文件根目录
```
